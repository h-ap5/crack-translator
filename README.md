### v2.6
- 모바일과 PC 모드 화면을 오갈 때 설정 단추가 화면 밖으로 사라지던 가출 버그를 고쳤습니다. 이제 화면 크기가 변해도 단추가 스스로 화면 안쪽으로 돌아옵니다.
